# 🚗 Sistema de Aluguel de Carros

Este projeto foi desenvolvido para prática de Software Products, criando um sistema simples que simula o funcionamento de uma locadora de veículos.

Aqui estão reunidos tanto o **Front-End** quanto o **Back-End**, se comunicando através de uma API.  
O objetivo principal é entender como uma aplicação completa funciona utilizando banco de dados real.

## 🎯 Objetivo do Projeto

📌 Criar um sistema onde o usuário possa:
- Cadastrar e se Logar no site
- Visualizar carros disponíveis para aluguel
- Escolher e realizar o aluguel de um veículo
- Devolver um carro
- Consultar os alugueis que já foram feitos
- Dash é criado com o total de carros alugados, disponiveis e devolvidos

## 🧰 Tecnologias Utilizadas

| **Front-End** : HTML • CSS • JavaScript |
| **Back-End** : Python + Flask |
| **Banco de Dados** : MySQL (Modelagem e gerenciamento via **MySQL Workbench**)
